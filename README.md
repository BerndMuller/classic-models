## Getting Started
```
mvn clean package
```
Dann deployen Sie wie gewohnt auf Ihrem Java-EE-8-Application-Server


### Voraussetzungen

Java-EE-8-Application-Server!

Wir haben mit WildFly 19 unter Java 13 getestet.
